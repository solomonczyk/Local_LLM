# SEO guide (local)

## 1) Technical SEO fundamentals
- Semantic HTML structure (h1, h2, nav, main, article)
- Meta tags: title (50-60 chars), description (150-160 chars)
- Open Graph tags for social sharing
- Canonical URLs to avoid duplicate content
- XML sitemap and robots.txt

## 2) Performance & Core Web Vitals
- Page load speed (< 3 seconds)
- Largest Contentful Paint (LCP < 2.5s)
- First Input Delay (FID < 100ms)
- Cumulative Layout Shift (CLS < 0.1)
- Mobile-first responsive design

## 3) Content optimization
- Keyword research and natural placement
- Clear heading hierarchy
- Alt text for images
- Internal linking structure
- Fresh, relevant content

## 4) Accessibility = SEO
- ARIA labels where needed
- Keyboard navigation
- Screen reader compatibility
- Color contrast ratios
- Descriptive link text

## 5) Common pitfalls
- JavaScript-only content (not crawlable)
- Missing or duplicate meta tags
- Broken links (404s)
- Slow server response times
- No mobile optimization

## 6) Output rules for SEO agent
Always return:
- SEO issues found (ranked by impact)
- Quick wins (easy improvements)
- Technical recommendations
- Content suggestions
- Monitoring metrics to track
